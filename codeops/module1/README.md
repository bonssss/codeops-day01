# Module 1 - Python Programming Practice

This module contains Python exercises and mini-projects covering core programming concepts such as:

- Variables, input/output, and basic operations
- Control flow with conditions and loops
- Functions and recursion
- Lists, dictionaries, sets, and tuples
- File handling and error handling
- Object-oriented programming (OOP)
- Data structures such as stacks, queues, linked lists, trees, graphs, and heaps
- Basic algorithm concepts such as searching and sorting

## Mini Project

The mini project in this module is the Addis Bank System.

It demonstrates:

- OOP concepts such as abstraction, encapsulation, inheritance, and polymorphism
- Design patterns such as Singleton, Factory, Observer, and Command
- Data structures including trees, graphs, heaps, and BSTs
- A console-based menu system for interacting with the program

## How to Run

From the project root, run:

```bash
python module1/miniproject/bank_system.py
```

## Running Tests

To run the tests for the mini project:

```bash
cd module1/miniproject
python -m pytest -q
```

## Learning Goals

By completing this module, you will practice:

- Writing clean Python programs
- Using classes and objects
- Applying data structures to solve problems
- Understanding common software design patterns
- Building small interactive applications
